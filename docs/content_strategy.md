# Content Strategy — Style Superman

把情報變成內容、把內容變成影響力。這份文件定義「我們為誰、做什麼、用什麼角度」。

## 1. 受眾

主受眾：**18–35 歲、對穿搭有興趣但不一定是硬核玩家的男性。**
他們想穿得更好、想跟上潮流，但被資訊量淹沒、也怕踩雷。我們替他們過濾、翻譯、示範。

次受眾：潮流玩家 / 同業——靠深度與觀點留住他們。

## 2. 內容支柱（Content Pillars）

| 支柱 | 佔比 | 說明 | 對應 content_angle |
|------|:----:|------|---------------------|
| 趨勢科普 | 30% | 「這個趨勢是什麼、從哪來」 | explainer |
| 實穿教學 | 30% | 「怎麼把趨勢穿上身」 | how-to-wear |
| 觀點 / 爭議 | 20% | 「這個趨勢值不值得跟」 | hot-take |
| 開箱 / 評測 | 10% | 單品實測 | haul-review |
| 盤點 / 清單 | 10% | 「本季 5 個必收」 | listicle |

刻意保留 20% 給 hot-take——反共識內容最能帶討論與記憶點，但不能過量否則變鍵盤俠。

## 3. 從情報到內容的轉換

```
Daily Brief 的 headline trend
        │
        ▼  prompts/short_video_ideas.md
2–3 個選題卡（含鉤子 / 分鏡 / CTA）
        │
        ▼  人工挑選 + 排程
拍攝 → 發布 → 回看數據 → 回饋給評分權重
```

## 4. 平台策略

| 平台 | 角色 | 內容形態 |
|------|------|---------|
| TikTok | 趨勢前沿、衝流量 | 15–30s 鉤子強的短影音 |
| Reels | 與 IG 生態整合、導購 | 同片二次利用 + 圖文 |
| Shorts | 補觸及、SEO 長尾 | explainer 類耐看內容 |
| IG 貼文 / 限動 | 經營鐵粉、即時 | brief 精華圖卡、品牌動態 |

一條趨勢 → 一支主片 → 多平台分發 + 圖文衍生。最大化單一情報的產出。

## 5. 調性（Brand Voice）

- **自信但不裝** — 有觀點，但不貶低觀眾。
- **實用優先** — 每支片要讓人「學到 / 想試 / 想買」其中之一。
- **快、準、有態度** — 不講廢話開場，前 3 秒給價值。
- **誠實** — 不為流量吹捧不值得的趨勢；該潑冷水就潑。

## 6. 節奏

- **每日**：產出 brief，挑 1 個趨勢進選題池。
- **每週**：從選題池排出 3–5 支拍攝。
- **每月**：回看哪類內容表現好，調整支柱佔比與評分權重。

Daily brief → 每週選題 → 每月速報 → 主題分析的完整生產線，見 [Content Calendar](content_calendar.md)。

## 7. 成功指標

| 層級 | 指標 |
|------|------|
| 內容 | 完播率、互動率、分享數 |
| 情報 | brief 命中率（被選題採用 / 後續真的爆的比例） |
| 成長 | 追蹤數、回訪、導購轉換 |

情報系統的終極考核：**我們是不是比別人早一步看到趨勢。** 用 `reports/daily/` 的歷史回測這件事。
