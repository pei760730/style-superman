# 2026 初夏(5–6 月)男裝趨勢 · 歐美 / 日本 / 韓國

> 產出日期：2026-06-05｜涵蓋：SS2026 初夏（5–6 月）男裝，歐美 / 日本 / 韓國三市場，從頭到腳 + 值得關注品牌。
> 方法：repo 量化榜（Lyst / StockX / KREAM / MUSINSA / Mercari）+ 三組並行媒體研究。**每節附原文來源（英文優先）**，可點進去看更細。
> 鐵則：標「待查」者為未取得可靠來源、未編造；平台量化榜反映大眾基本盤，尖端風向看媒體街拍與名人私服，兩者分開讀。

## 本季一句話

**三地最大的共同故事 = 廓形從「極端寬鬆 / oversize」往回收，直筒、乾淨比例、正裝感回潮。**
韓國最決斷（wide 退、straight / barrel 丹寧 + slim 針織回流），日本次之（A-line 退、I-line 直筒回歸），歐美則是「下身放量 + 上身收乾淨」的比例遊戲。

---

## 從頭到腳

### 1. 頭
- **歐美**：草帽 / 巴拿馬帽（夏季第一順位）、漁夫帽材質升級、報童帽回潮；材質主軸 linen。髮型 **待查**。
- **日本**：經典帽回歸（fedora / 獵帽）、深版 + 長簷 cap；New Era 當紅、ARC'TERYX tech cap。帽市分三型（奢侈 status / 機能 tech / 韓系 silhouette）。
- **韓國**：Cap / Beanie 疊搭基本盤；**眼鏡成頭部最關鍵配件**（Poet Core 帶動復古光學框）。

### 2. 上半身（T / 襯衫 / Polo / 針織）
- **歐美**：**Sporty-Prep 運動學院風**主軸（Miu Miu / Lacoste / Prada）——條紋 **Rugby 橄欖球衫**、粗針織 Polo；街拍 oversized 長版襯衫（深 V、亮面、70–80s 印花）。
- **日本**：**通透感襯衫**（sheer / 開領 / 睡衣領）當主角；圖樣以條紋為軸；背心當可見內搭 layering；版型偏 box，但身長略長的貼身 cut&sewn 開始冒頭。
- **韓國**：**雙軌**——slim fit 針織回流（秀身形線條）+ 日常 oversized 襯衫鬆領帶；圖樣是「적토마（赤兔馬）」強烈紅色動態 graphic tee。
- 🔑 **Rugby 橄欖球衫**在歐美與韓國同步出現，跨市場可下注。

### 3. 外套（夏季）
三地一致：**工裝 / 軍裝大回歸 + 薄西裝（setup）休閒化。**
- **歐美**：Regiment 軍團大衣全面回歸（Ann Demeulemeester 復刻 / Craig Green 解構）、soft tailoring 軟結構薄西裝、Dior（JW Anderson 首秀）超大 cargo。
- **日本**：短版 zip-up blouson（更短更 compact）貫穿春夏、authentic 軍 / 工裝（M-65 / CPO / coverall）、輕量化 setup。
- **韓國**：丹寧 trucker 夾克取代西裝外套、workwear 工裝（做舊感當裝飾）、麂皮 / 羊皮輕外套、「一套搞定」setup 最高分。

### 4. 下半身（長褲 / 短褲 / 丹寧）— 本季最大轉折
- **長褲廓形**：skinny 早死，**wide 也開始退**。歐美仍寬腿（下鬆上合）；日本「寬版量感收斂、回 I-line 直筒」；**韓國最決斷——「wide / baggy 時代過去，straight 時間到來」**，丹寧熱詞 **Barrel Jeans（桶形）**、Classic Straight、Bootcut（褲口微喇）。
- **丹寧質感**：歐美 baggy 但比例乾淨（黑 / 灰 / 米 / 白）；日本偏做舊 fade；韓國 destroyed + barrel 立體剪裁。
- **短褲（強訊號）**：**Micro Shorts 超短褲**在歐美與韓國同步從伸展台（Prada / Saint Laurent）冒出；日本是「低腰短褲 + tailoring + 涼鞋」dress-resort 街拍。Cargo 短褲三地都在。

### 6. 配件
- **包**：歐美 slouchy 郵差包（ARCS / Master-Piece）；日本機能斜背小包、**Prada Re-Nylon 人氣 No.1**、異素材 mix；韓國 **Long Handle Bag（細長把手）**、East-West 橫長包、utility bag。
- **眼鏡 / 太陽眼鏡**：歐美大方厚框 statement；**韓國 Gentle Monster 2026 Bouquet Collection** + Poet Core 復古光學框；日本太陽眼鏡 **待查**。
- **銀飾（三地一致看漲）**：Silver 強勢回歸。歐美串珠 + 極簡素銀（≤2 件）；日本霧面厚重感（Lion Heart / Chrome Hearts）；韓國 **Chrome Hearts** 熱（曾與 2NE1 聯名、Cemetery Ring），呼應 Romantic Goth。
- **襪子**：三地都出現涼鞋配襪；歐美 / 韓國蓬鬆 crew socks。

### 7. 顏色 / 材質 / 廓形
- **顏色**：三地底色都是大地中性 + 洗舊感（米 / 沙 / 灰 / 橄欖）。亮點色：歐美 buttery yellow / soft teal / burnt orange；日本 yellow / pink / mint +「紅」；韓國 **적토마紅** 年度標誌色 + Poet Core 象牙 / 褪黑。
- **材質**：**linen / 亞麻三地共同主力** + seersucker、summer corduroy、再生 nylon、麂皮。
- **廓形總結**：oversize 與 soft tailored 並存，但天平往「乾淨比例 / 正裝感回潮」傾——本季最該記住的一句。
